tor and privoxy

Category: tools Tags: tor privoxy

Tor: due to vps centos have no offical reponsity,we need to create ourself:

vi /etc/yum.repos.d/torproject.repo

[torproject]
name=Tor and Vidalia
enabled=1
autorefresh=0
#baseurl=http://deb.torproject.org/torproject.org/rpm/OBSOLETE/centos5/
baseurl=http://deb.torproject.org/torproject.org/rpm/el/6/i686/
type=rpm-md
gpgcheck=1
gpgkey=http://deb.torproject.org/torproject.org/rpm/RPM-GPG-KEY-torproject.org

[dag]
name=Dag RPM Repository for Red Hat Enterprise Linux
baseurl=http://apt.sw.be/redhat/el$releasever/en/$basearch/dag
gpgcheck=1
enabled=1
gpgkey=http://dag.wieers.com/rpm/packages/RPM-GPG-KEY.dag.txt

when i setup tor on centos6 ,then failed.

above maybe work for centos5 , but update:

[tor]
name=tor
enabled=1
baseurl=http://deb.torproject.org/torproject.org/rpm/el/5/i386/
gpgcheck=1
gpgkey=http://deb.torproject.org/torproject.org/rpm/RPM-GPG-KEY-torproject.org.EL5.asc

[tor-source]
name=Tor experimental source repo
enabled=1
autorefresh=0
baseurl=http://deb.torproject.org/torproject.org/rpm/el/5/SRPMS
gpgcheck=1
gpgkey=http://deb.torproject.org/torproject.org/rpm/RPM-GPG-KEY-torproject.org.EL5.asc

then:

yum install tor /etc/init.d/tor start

Privoxy:

yum install gcc autoconf
groupadd privoxy
useradd -g privoxy privoxy
wget http://sourceforge.net/projects/ijbswa/files/Sources/3.0.19%20%28stable%29/privoxy-3.0.19-stable-src.tar.gz/download
tar zxvf privoxy-3.0.19-stable-src.tar.gz
cd privoxy-3.0.19-stable

make
make install

#add this to /usr/local/etc/privoxy/config
echo “forward-socks4a / 127.0.0.1:9050 . ” >> /usr/local/etc/privoxy/config

should be work.
